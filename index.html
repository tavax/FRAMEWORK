Les concepts de base de l'utilisation du framework

Où récupérer les sources ?

Un dépot GIT Cabestan a été mis en place et tout y est accessible d'un simple download.

Comment se connecter ?

Il faut se rendre à l'adresse du dépot disponible ici et cliquer sur le bouton pour y récupérer la ligne suivante :

      									https://github.com/tavax/FRAMEWORK.git

Comment utiliser cette URL ?

l'extension .git fonctionne en ligne de commande. Pour que cela soit utilisable, il faut installer sur le poste de travail plusieurs choses :

  • Télécharger et installer node.js (nécessite les droits admin de la machine pour son installation).
  • Télécharger et installer Git.

Mise en place de l'environnement de travail

Une fois les sources en place et node.js et git installer, il faut récupérer les dépendances des scripts. un clic droit dans le dossier et sélectionner la ligne "Git Bash Here".

dans la fenêtre qui va s'ouvrir, la ligne de commande suivante va s'en charger.

                                 npm install

Procédure de travail (nouveau dépot)

Le dossier est à jour.

La première chose à faire et le plus important : la création de branche. Rien de plus simple, la commande suivant vas vous montrer toutes les branches que vous avez réalisée

                                 git branch

On ne travaille JAMAIS sur la branche master d'un dépot, cette branche est votre backup de secours, elle doit toujours être à jour par rapport au dépot distant.

Procédure de travail (mettre à jour la branche master)

La ligne de commande suivante vous permets de faire un état des lieux du code, à la fois en local et distant du dépot

                                 git status
