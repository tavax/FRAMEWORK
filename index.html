Les concepts de base de l'utilisation du framework

Où récupérer les sources ?

Un dépot GIT Cabestan a été mis en place et tout y est accessible d'un simple download.

Comment se connecter ?

Il faut se rendre à l'adresse du dépot disponible ici et cliquer sur le bouton pour y récupérer la ligne suivante :

									https://github.com/tavax/FRAMEWORK.git

Comment utiliser cette URL ?

l'extension .git fonctionne en ligne de commande. Pour que cela soit utilisable, il faut installer sur le poste de travail plusieurs choses.
